## Установка со стоковой прошивки

::: warning
Роутер должен быть подключён через кабель. Отключите брандмауэр, антивирус вашей ОС, а так же остальные проводные подключения, если они есть. При возникновении проблем используйте `Безопасный режим с поддержкой сети`.
:::

1. Во вкладке `Дополнительные настройки` выберите пункт `Резервн. копия` и загрузите файл конфигурации **settings_@keeneticported.bin**.
![альтернативный текст](/assets/images/wiki/guides/WBR3000UAX/1.jpg)
2. После перезагрузки роутера запустите файл **wbr_installer.exe**, дождитесь завершения работы скрипта, после которого роутер перезапустится и прошивка будет установлена.

::: tip Внимание
Файл пропатчит EEPROM, сгенерирует новые сервисные цифры и установит прошивку.
:::

3. После перезагрузки устройство в течение 3-х минут запустится в KeeneticOS
::: tip URL: 192.168.1.1 `admin:12345678`<br/>SSID: Keenetic<br/>Password: 12345678
:::

## Установка с OpenWRT прошивки

::: danger ВНИМАНИЕ
Не используйте загрузчик OpenWRT, необходим родной.
:::

1. В разделе `System - Backup / Flash firmware` сделать бэкапы всех разделов
2. Из под веб-интерфейса OpenWRT (или веб-интерфейса загрузчика) загрузить `Keenetic-Cudy_WBR3000UAX.bin` как
   обновление c галочкой `Force Upgrade`.
::: tip Внимание
Файл пропатчит EEPROM, сгенерирует новые сервисные цифры и установит прошивку.
:::
3. После перезагрузки устройство в течение 3-х минут запустится в KeeneticOS
::: tip URL: 192.168.1.1 `admin:12345678`<br/>SSID: Keenetic<br/>Password: 12345678
:::

## Откат на стоковую прошивку

1. Зайти в загрузчик KeenBOOT зажатием кнопки Reset
2. В разделе `Обновление` загрузить бэкап `mtd0_spi0.0.bin` как `Полный образ (Full)`, полученный скриптом при первой установке.

Для Cudy WBR3000UAX есть два варианта USB-мода

### Простой

Удалить отмеченный резистор R302

![альтернативный текст](/assets/images/wiki/helpful/usb-mod/wbr3000uax-1.jpg){width=300px height=100px}

![альтернативный текст](/assets/images/wiki/helpful/usb-mod/wbr3000uax-2.jpg){width=300px height=100px}

### Поднятие тока на порту до 2А

Для этого меняем отмеченный резистор R304 на новый с номиналом 2.7 кОм

![альтернативный текст](/assets/images/wiki/helpful/usb-mod/wbr3000uax-3.jpg){width=300px height=100px}
