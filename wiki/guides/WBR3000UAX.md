## Подготовка к прошивке

::: warning
Роутер должен быть подключён к ПК кабелем. Отключите брандмауэр и антивирус вашей ОС, а также остальные проводные подключения, если они есть. При возникновении проблем используйте `Безопасный режим с поддержкой сети`.
:::

::: tip О скрипте-прошивальщике
Файл `Cudy_WBR3000UAX_flasher.exe` создаёт бэкапы, патчит EEPROM, генерирует новые сервисные цифры и устанавливает прошивку, после чего роутер автоматически перезагружается.
:::

## Установка со стоковой прошивки

1. **Загрузка конфигурации.** Во вкладке `Дополнительные настройки` выберите пункт `Резервн. копия` и загрузите файл конфигурации `settings_@keeneticported.bin`.

   ![Загрузка файла конфигурации](/assets/images/wiki/guides/Cudy/flash.jpg)

2. **Прошивка.** После перезагрузки запустите `Cudy_WBR3000UAX_flasher.exe`, выберите вариант **Стоковая прошивка**, нажмите «Установить» и дождитесь завершения.
3. **Загрузка системы.** В течение ~3 минут устройство запустится в KeeneticOS.

::: tip Готово! Доступ к роутеру
URL: `192.168.1.1` (`admin` / `12345678`)<br/>SSID: `Keenetic`<br/>Пароль Wi-Fi: `12345678`
:::

## Установка с OpenWrt

::: danger ВНИМАНИЕ
Способ актуален только для стандартной OpenWrt — U-Boot Layout не подойдёт!
:::

1. **Загрузка образа.** В веб-интерфейсе OpenWrt (или веб-интерфейсе загрузчика) загрузите `default-openwrt.bin` как обновление. Галочку `Сохранить конфигурацию` можно убрать.
2. **Прошивка.** После перезагрузки запустите `Cudy_WBR3000UAX_flasher.exe`, выберите вариант **OpenWrt**, нажмите «Установить» и дождитесь завершения.
3. **Загрузка системы.** В течение ~3 минут устройство запустится в KeeneticOS.

::: tip Готово! Доступ к роутеру
URL: `192.168.1.1` (`admin` / `12345678`)<br/>SSID: `Keenetic`<br/>Пароль Wi-Fi: `12345678`
:::

## Откат на стоковую прошивку

1. Зайдите в загрузчик KeenBOOT зажатием кнопки Reset.
2. В разделе `Обновление` загрузите бэкап `mtd0_spi0.0.bin` как `Полный образ (Full)` — он был получен скриптом при первой установке.

![Откат прошивки через KeenBOOT](/assets/images/wiki/guides/Cudy/revert.png)

## Перезагрузка USB-модема (USB-Mod) {#usb-mod}

::: info
Мод нужен для перезагрузки USB-модема по питанию — [подробнее](https://help.keenetic.com/hc/ru/articles/115000041605-%D0%9C%D0%BE%D0%B6%D0%BD%D0%BE-%D0%BB%D0%B8-%D0%BF%D0%B5%D1%80%D0%B5%D0%B7%D0%B0%D0%B3%D1%80%D1%83%D0%B7%D0%B8%D1%82%D1%8C-%D0%BF%D0%BE-%D0%BF%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8E-USB-%D0%BC%D0%BE%D0%B4%D0%B5%D0%BC-%D0%BF%D0%BE%D0%B4%D0%BA%D0%BB%D1%8E%D1%87%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9-%D0%BA-Keenetic-%D1%81%D1%80%D0%B5%D0%B4%D1%81%D1%82%D0%B2%D0%B0%D0%BC%D0%B8-%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82-%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0)
:::

::: warning
Новая ревизия (`SN: S00`) поддерживает перезагрузку USB-модема «из коробки» — модификация не требуется!
:::

### Два варианта USB-мода

1. **Простой.** Удалите отмеченный резистор `R302`.

   ![USB-Mod WBR3000UAX, простой вариант](/assets/images/wiki/helpful/usb-mod/wbr3000uax-1.jpg){width=300px height=100px}

2. **Сложный.** Замените отмеченный резистор `R305` новым с номиналом 3.3 кОм.

   ![USB-Mod WBR3000UAX, сложный вариант](/assets/images/wiki/helpful/usb-mod/wbr3000uax-2.jpg){width=300px height=100px}

### Поднятие тока на порту до 2 А

Замените отмеченный резистор `R304` новым с номиналом 2.7 кОм.

![USB-Mod WBR3000UAX, поднятие тока](/assets/images/wiki/helpful/usb-mod/wbr3000uax-3.jpg){width=300px height=100px}
