## Подготовка к прошивке

::: danger Перед прошивкой убедитесь
- Версия стоковой прошивки на роутере — **ниже 1.0.90**.
- Дата производства подходит для прошивки (см. таблицу ниже). Роутеры с неподходящей датой **не поддерживаются** и поддерживаться не будут.
- Поддерживаются обе модели памяти — **ESMT** и **Winbond**, проверять установленную память не нужно.
- Всю ответственность за устройство несёте только вы.
:::

**1. Проверьте дату производства** (наклейка на дне роутера):

| Версия устройства | Допустимая дата выпуска     |
| ----------------- | --------------------------- |
| 🇨🇳 Китайская      | до **2024.06** включительно |
| 🌍 Глобальная     | до **2024.12** включительно |

**2. Скачайте и распакуйте архив с прошивкой** — кнопка с версией в заголовке страницы.

**3. Подключите роутер к компьютеру кабелем (Ethernet).** Прошивка по Wi-Fi не выполняется.

## Установка со стоковой прошивки

1. **Сброс настроек.** Сбросьте роутер до заводских настроек.

2. **Первоначальная настройка.** Пройдите мастер настройки и задайте пароль для входа в веб-интерфейс — запомните его.

3. **Первый скрипт.** Запустите `1_ax3000t.exe`, введите заданный пароль и дождитесь завершения. После этого роутер перезагрузится.

4. **Второй скрипт.** Дождитесь загрузки роутера, восстановите проводное подключение и запустите `Xiaomi_AX3000T_flasher.exe`. Нажмите «Установить» и дождитесь завершения — роутер автоматически перезагрузится уже с новой прошивкой.

::: tip Готово! Доступ к роутеру
URL: `192.168.1.1` (`admin` / `12345678`)<br/>SSID: `Keenetic`<br/>Пароль Wi-Fi: `12345678`
:::

::: tip Снижение потребления ОЗУ (опционально)
Чтобы уменьшить расход оперативной памяти, можно отключить аппаратное ускорение WHNAT — [подробнее](https://help.keenetic.ru/hc/ru/articles/214470905.html). Команды вводятся в CLI роутера по очереди:

```shell
interface WifiMaster1 no whnat
no ppe hardware
system configuration save
```
:::
