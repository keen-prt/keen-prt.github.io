## Подготовка к прошивке

Скачайте и распакуйте архив с прошивкой (кнопка с версией в заголовке страницы) и подключите роутер к ПК кабелем (Ethernet).

## Установка со стоковой прошивки

1. **Сброс и настройка.** Сбросьте роутер, пройдите первоначальную настройку и задайте пароль для входа в веб-интерфейс.
2. **Запуск скрипта.** Запустите `Netis_NX32U_flasher.exe` и укажите заданный пароль.
3. **Резервные копии.** Перед установкой в папке со скриптом создадутся бэкапы системы — сохраните их в надёжном месте.
4. **Прошивка.** Подтвердите установку прошивки и дождитесь завершения.

После перезагрузки устройство запустится в KeeneticOS.

::: tip Готово! Доступ к роутеру
URL: `192.168.1.1` (`admin` / `12345678`)<br/>SSID: `Keenetic`<br/>Пароль Wi-Fi: `12345678`
:::

::: tip Снижение потребления ОЗУ (опционально)
Чтобы уменьшить расход оперативной памяти, можно отключить аппаратное ускорение WHNAT — [подробнее](https://help.keenetic.ru/hc/ru/articles/214470905.html). Команды вводятся в CLI роутера по очереди:

```shell
interface WifiMaster1 no whnat
no ppe hardware
system configuration save
```
:::
