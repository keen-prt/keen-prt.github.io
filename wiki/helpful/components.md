# Установленные компоненты
::: tip Внимание
Актуально для устройств со статусом `Active` и `KeeneticOS 4.2.1+`
:::

- English
- Русский

- Интерфейс Wi-Fi
- Интерфейс USB
- Сетевой ускоритель
- Сервер DHCP
- Прокси-службы для IGMP и PPPoE
- Служба UPnP
- Сервер SSH
- Контроллер Wi-Fi-системы
